# GhostEditor

## Setup Instructions

### Prerequisites
- elixir
- erlang
- python3.10

### Setup Instructions

1. **Clone the repository**
  ```bash
  git clone https://github.com/suvanshenoy/ghost-editor.git
  cd ghost-editor
  ```

2. **Run the project**
  ```
  mix deps.get
  mix run --no-halt

  ```
